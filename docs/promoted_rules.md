**최근 사례 (날짜별 rule)**
- **2026-04-28** 전일 +7% 이상 급등 + 당일 추가 촉매 부재 → 관망이 아닌 하락 경계(sell) 고려.
- **2026-04-30** 뉴스 부정 + 공매도 지정 혹은 가격제한폭 도달 등 구조적 약세 신호가 겹치면 관망 대신 최소 하락 경계로 격상.
- **2026-05-01** 공매도 과열 지정 + 부정 뉴스 평가가 동시에 존재하면 다음날 기본 판정을 하락 경계로 설정; 낙폭 과대를 근거로 한 관망 설정 금지

**반복 횟수**: 7일 (2026-04-21, 2026-04-24, 2026-04-27, 2026-04-28, 2026-04-30, 2026-05-01, 2026-05-04)

- **2026-05-01** 공매도 과열 지정 + 부정 뉴스 평가가 동시에 존재하면 다음날 기본 판정을 하락 경계로 설정; 낙폭 과대를 근거로 한 관망 설정 금지
- **2026-05-04** 바이오 ETF 약세 + 소형 바이오 + 24시간 내 개별 뉴스 없음 조합에서는 하락 경계(sell) 판단을 우선 검토할 것

## 3. 선반영 과소평가 방지

**적용 규칙**
> 지수 +3% 이상 예상일에는 선반영 판단 기반 관망 종목도 '소폭 상승' 방향 편향을 유지하고 강한 매도는 자제할 것

**반복 횟수**: 6일 (2026-04-21, 2026-04-23, 2026-04-27, 2026-04-29, 2026-04-30, 2026-05-04)

**최근 사례 (날짜별 rule)**
- **2026-04-29** 전일 -1% 이상 하락 + 간밤 프록시 급락(-3% 이상) → 강한 하락 경계 대신 하락 경계(1단계 완화) 적용
- **2026-04-30** 대형주 실적 확정 발표 당일: 뉴스 긍정이어도 전일 이미 +1% 이상 상승했다면 선반영 가능성 기본값으로 관망 처리.
- **2026-05-04** 지수 +3% 이상 예상일에는 선반영 판단 기반 관망 종목도 '소폭 상승' 방향 편향을 유지하고 강한 매도는 자제할 것

**배경**
삼성전자(선반영+파업 → 관망 → 실제 +5.44%)처럼, 개별 종목이 선반영 상태여도 지수 전체가 5% 급등하면 베타에 의해 강제 상승한다. 선반영 판단은 지수 중립일 기준이며 지수 급등일에는 수정이 필요하다.

## 4. 섹터 지수 우선 (SOX·XBI·XLE 등)

**적용 규칙**
> overnight 라벨이 '강세'이더라도 구성 핵심주(NVDA, META 등)가 -3% 이상이면 해당 섹터 방향을 중립 또는 하향 조정할 것

**반복 횟수**: 4일 (2026-04-28, 2026-04-29, 2026-04-30, 2026-05-01)

**최근 사례 (날짜별 rule)**
- **2026-04-29** 전일 -1% 이상 하락 + 간밤 프록시 급락(-3% 이상) → 강한 하락 경계 대신 하락 경계(1단계 완화) 적용
- **2026-04-30** 개별 프록시가 혼조일 때는 상위 섹터 지수(SOX, XBI, XLE 등) 방향을 기본 신호로 채택.
- **2026-05-01** 시장 방향이 혼조·불확실하거나 overnight 신호가 엇갈릴 때 레버리지 ETF 상승 기대/강한 상승 기대 예측 금지; 최소 관망으로 설정

**배경**
SOX +2.26%가 '강세'로 집계됐어도 구성 내 NVDA -4.63%가 있으면 실질 반도체 센티멘트는 약세다. 라벨이 아닌 핵심 개별 프록시 수치를 직접 확인해야 한다. 오늘 overnight_misled 12건의 근본 원인이 이것이었다.
